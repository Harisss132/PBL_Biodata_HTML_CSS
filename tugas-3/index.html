About Us

Saya adalah peserta pelatihan project base learning di BBPVP Bekasi. Saya memilih jurusan pengembangan web dengan node.js dan react, saya tertatik dengan teknologi terutama di bidang pengembang web back-end, dan juga saya sedang belajar untuk sisi front-ednya, semoga saya bisa konsiten.

Alasan saya mengikuti PBL ini karena minat saya dalam bidang pengembangan web, dan juga karena ada OJT saya berharap dengan adanya pelatihan seperti ini yang bisa langsung menerapkan real studi case terutama dalam industri, saya bisa membuat portofolio yang mendukung dengan adanya OJT.
